## Usage

Install:

```bash
npm add rspress-plugin-font-open-sans -D
```

Add plugin to your `rspress.config.ts`:

```ts
// rspress.config.ts
import { pluginFontOpenSans } from 'rspress-plugin-font-open-sans';

export default {
  plugins: [pluginFontOpenSans()],
};
```
